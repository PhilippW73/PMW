Psychic-Game

The app randomly picks a letter and the user has to guess which letter the app chose. The user has 9 guesses to match the randomly computer picked letter. When the player wins, the Wins-Counter will be increased and the game starts over again (without refreshing the page). When the player loses, the Losses-Counter will be increased and the game (after 9 guesses) restarts without a page refresh (just like when the user wins).

Technologies used: HTML / CSS / JAVASCRIPT / Bootstrap
